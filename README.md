**Интерпритатор языка прогрммирования Mython.**


**Описание**

Задумывался как учебный проект, для того что бы попытать свои силы в написании простых ЯП на С++17.
Думаю получилось неплохо)).

В проекте все написано с нуля лексический, синтаксический, симатический анализаторы, таблица символов.
Все это сделано для более глубокого понимания как устроены интерпритаторы.

**Сборка**

Сборка проекта более чем тривиальна, достаточно скопировать репозиторий и собрать всем чем душе угодно)).
